Animation 속성

속성 입력값/선택값 설명
animation-name 명칭입력 애니메이션 키프레임 이름
animation-duration 초단위 입력 애니메이션 1회의 진행 시간
animation-timing-function linear, ease, ease-in, ease-out, ease-in-out 등 재생 중간의 속도 지정
animation-delay 초단위 입력 이벤트 발생 후 몇 초 후에 애니메이션할지 지정
animation-iteration-count 횟수입력 혹은 infinite 반복횟수
animation-direction - normal : 기본 값 (forwards)
- reverse : 애니메이션이 역방향으로 진행 (backwards)
- alternate : 애니메이션이 기본값으로(앞) 진행된 후 역방향으로 번갈아 진행
- alternate-reverse : 애니메이션이 역방향으로(뒤) 진행된 후 앞으로 번갈아 진행		 애니메이션의 진행방향 지정
animation-fill-mode - none : 기본값 (따로 적용되지 않음)
- forwards : 마지막 키 프레임에 의해 설정된 스타일 값을 유지
- backwards : 첫 번째 키 프레임에 설정된 스타일 값을 가져오고
애니메이션 지연 기간 동안이 값을 유지
- both : 애니메이션은 앞뒤 모두에 대한 규칙을 따름		 애니메이션 시작 전, 끝나기 전(또는 둘 다) 스타일을 지정
animation-play-state - paused : 애니메이션 일시중지
- running : 기본값 (애니메이션 실행) 		재생상태 지정
animation name duration timing-function delay iteration-count direction fill-mode play-state 애니메이션 속성을 한꺼번에 지정
